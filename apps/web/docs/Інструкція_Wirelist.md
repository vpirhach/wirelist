# Детальне керівництво по використанню додатку Wirelist

## Огляд додатку

Wirelist є веб-додатком для управління переліком дротів та кабелів в технічних системах. Додаток дозволяє створювати, редагувати, видаляти записи про дроти та відстежувати історію змін.

## Вхід до системи

Перейдіть за адресою: `http://172.18.12.72/`

![Головна сторінка Wirelist](wirelist-main-page.png)

## Основні компоненти інтерфейсу

### Верхня панель навігації

- **Логотип "Wirelist Admin"** - повертає на головну сторінку
- **Поле пошуку** - "Пошук або введіть команду..." для швидкого знаходження записів
- **Кнопка "Створити +"** - для додавання нового дроту
- **Кнопка "Завантажити"** - для завантаження даних
- **Перемикач мови** - EN/UK для зміни мови інтерфейсу
- **Профіль користувача** - показує поточного користувача

### Бічна навігація

- **Список** - основна таблиця всіх дротів
- **Список змін** - історія всіх операцій зі зміни дротів

## Основні функції

### 1. Перегляд списку дротів

![Таблиця дротів](wirelist-table-view.png)

При натисканні на пункт меню "Список" відображається головна таблиця з наступними колонками:

- **Від** - початкова точка з'єднання
- **До** - кінцева точка з'єднання
- **Код проводу** - технічний код дроту
- **Код кольору** - код кольору дроту
- **Код примітки** - додаткові технічні коди
- **Тип I/O** - тип вводу/виводу
- **Підконтролер** - номер підконтролера
- **Слово** - номер слова в контролері
- **Біт** - номер біту
- **Примітки** - текстовий опис дроту
- **Живлення** - тип живлення
- **Джерело** - джерело сигналу
- **Пед** - педагогічні дані
- **Мережа** - мережні параметри
- **Дії** - кнопки редагування та видалення

#### Розширена таблиця

Використовуйте перемикач "Розширена таблиця" для відображення всіх доступних колонок.

### 2. Пошук записів

![Результати пошуку](wirelist-search-results.png)

Для пошуку дротів:

1. Введіть пошуковий запит у поле "Пошук або введіть команду..."
2. Натисніть Enter
3. Таблиця автоматично відфільтрується відповідно до запиту

**Приклад:** Введення "01A1" покаже лише записи, що містять цей текст.

### 3. Створення нового дроту

![Форма створення дроту](wirelist-create-form.png)

Для створення нового дроту:

1. Натисніть кнопку **"Створити +"** у верхній панелі
2. Заповніть форму з наступними розділами:

#### Розділ "Призначення":

- **Від** - вкажіть початкову точку з'єднання
- **До** - вкажіть кінцеву точку з'єднання

#### Розділ "Адреса":

- **Підконтролер** - номер підконтролера (за замовчуванням 0)
- **Слово** - номер слова (за замовчуванням 0)
- **Біт** - номер біту (за замовчуванням 0)

#### Розділ "Інше":

- **Код проводу** - технічний код
- **Код кольору** - код кольору
- **Код примітки** - додаткові коди
- **Тип I/O** - тип вводу/виводу
- **Примітки** - текстовий опис
- **Живлення** - тип живлення
- **Джерело** - джерело сигналу
- **Мережа** - мережні параметри

3. Натисніть **"Зберегти"** для збереження або **"Скасувати"** для відміни

![Заповнена форма](wirelist-filled-form.png)

### 4. Редагування існуючого дроту

![Форма редагування](wirelist-edit-form.png)

Для редагування дроту:

1. У таблиці знайдіть потрібний запис
2. Натисніть кнопку редагування (іконка олівця) у колонці "Дії"
3. Змініть необхідні поля в формі
4. Натисніть **"Зберегти"** для збереження змін або **"Скасувати"** для відміни

### 5. Видалення дроту

Для видалення дроту:

1. У таблиці знайдіть потрібний запис
2. Натисніть кнопку видалення (іконка кошика) у колонці "Дії"
3. Підтвердіть видалення

### 6. Перегляд історії змін

![Список змін](wirelist-changes-list.png)

Для перегляду історії операцій:

1. Натисніть пункт меню **"Список змін"**
2. Переглядайте таблицю з наступними колонками:
   - **Тип** - тип операції (CREATE, UPDATE, DELETE)
   - **Дата** - дата та час операції
   - **Автор** - користувач, що виконав операцію
   - **Від/До** - точки з'єднання
   - Інші технічні параметри дроту

Це дозволяє відстежувати всі зміни, хто і коли їх вніс.

### 7. Пагінація

Використовуйте елементи управління пагінацією внизу таблиці:

- Номери сторінок для переходу до конкретної сторінки
- Вибір кількості записів на сторінці (30/50 на сторінку)
- Навігаційні стрілки для переходу між сторінками

## Корисні поради

1. **Використовуйте пошук** для швидкого знаходження потрібних дротів
2. **Перевіряйте "Список змін"** для контролю модифікацій системи
3. **Заповнюйте поле "Примітки"** для кращого розуміння призначення дроту
4. **Використовуйте стандартизовані коди** у полях для консистентності даних

## Технічні особливості

- Додаток працює в режимі реального часу
- Всі зміни автоматично зберігаються в історії
- Підтримка українського інтерфейсу
- Можливість перемикання між мовами
- Адаптивний дизайн для різних розмірів екрану

## Заключення

Wirelist є потужним інструментом для управління електричними з'єднаннями. Використовуйте всі доступні функції для ефективного ведення документації по дротам та кабелям у ваших технічних системах.
