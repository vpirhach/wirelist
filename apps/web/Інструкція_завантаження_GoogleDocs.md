# Як завантажити керівництво Wirelist у Google Docs

## Підготовлені файли

### 📄 Текстовий файл

- `Інструкція_Wirelist_GoogleDocs.txt` - готовий текст без Markdown форматування

### 🖼️ Скріншоти для вставки

1. `wirelist-main-page.png` - Головна сторінка
2. `wirelist-table-view.png` - Таблиця дротів
3. `wirelist-search-results.png` - Результати пошуку
4. `wirelist-create-form.png` - Форма створення
5. `wirelist-filled-form.png` - Заповнена форма
6. `wirelist-edit-form.png` - Форма редагування
7. `wirelist-changes-list.png` - Список змін

## 📋 Покрокова інструкція завантаження

### Варіант 1: Копіювання тексту

1. **Відкрийте Google Docs**

   - Перейдіть на https://docs.google.com
   - Натисніть "Створити новий документ"

2. **Скопіюйте текст**

   - Відкрийте файл `Інструкція_Wirelist_GoogleDocs.txt`
   - Виділіть весь текст (Ctrl+A / Cmd+A)
   - Скопіюйте (Ctrl+C / Cmd+C)

3. **Вставте у Google Docs**
   - Вставте текст у документ (Ctrl+V / Cmd+V)

### Варіант 2: Завантаження файлу

1. **Завантажте файл у Google Drive**

   - Перейдіть на https://drive.google.com
   - Натисніть "Новий" → "Завантажити файл"
   - Виберіть `Інструкція_Wirelist_GoogleDocs.txt`

2. **Відкрийте у Google Docs**
   - Знайдіть завантажений файл у Drive
   - Клікніть правою кнопкою → "Відкрити за допомогою" → "Google Документи"

## 🖼️ Вставка скріншотів

### Розташування скріншотів у тексті

У тексті знайдіть наступні позначки та замініть їх зображеннями:

```
[СКРІНШОТ 1: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-main-page.png" - Головна сторінка Wirelist]
[СКРІНШОТ 2: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-table-view.png" - Таблиця дротів]
[СКРІНШОТ 3: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-search-results.png" - Результати пошуку]
[СКРІНШОТ 4: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-create-form.png" - Форма створення дроту]
[СКРІНШОТ 5: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-filled-form.png" - Заповнена форма]
[СКРІНШОТ 6: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-edit-form.png" - Форма редагування]
[СКРІНШОТ 7: ВСТАВИТИ ЗОБРАЖЕННЯ "wirelist-changes-list.png" - Список змін]
```

### Як вставити зображення

1. **Позиціонуйте курсор** на місце скріншота
2. **Видаліть текст позначки** (наприклад, `[СКРІНШОТ 1: ...]`)
3. **Вставте зображення**:
   - Меню "Вставка" → "Зображення" → "Завантажити з комп'ютера"
   - Або перетягніть файл зображення у документ
4. **Налаштуйте зображення**:
   - Клікніть на зображення
   - Виберіть "Обтікання тексту" → "Розривати текст"
   - Змініть розмір за потреби

## 🎨 Форматування документа

### Заголовки

- **Основний заголовок**: Формат → Стилі абзаців → "Заголовок 1"
- **Розділи**: "Заголовок 2"
- **Підрозділи**: "Заголовок 3"

### Списки

- Виділіть текст зі знаками "•"
- Натисніть кнопку "Маркований список" на панелі інструментів

### Жирний текст

- Виділіть текст у лапках або що потребує виділення
- Натисніть Ctrl+B / Cmd+B

### Моноширинний шрифт (для адрес)

- Виділіть `http://172.18.12.72/`
- Формат → Шрифт → "Courier New" або "Consolas"

## ✅ Фінальні кроки

1. **Перевірте документ**

   - Переконайтеся, що всі скріншоти вставлені
   - Перевірте форматування заголовків та списків

2. **Додайте назву документа**

   - Клікніть на "Документ без назви"
   - Введіть: "Керівництво по використанню Wirelist"

3. **Налаштуйте доступ**
   - Натисніть "Надати доступ" у правому верхньому куті
   - Налаштуйте права доступу для команди
